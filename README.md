# 🚀 Focus90 | Career Mastery Tracker

**Focus90** is a high-performance productivity dashboard designed to help developers and students become "Job Ready" in 90 days. It tracks daily tasks, learning hours, and provides visual analytics to ensure consistent progress toward career goals.

---

## ✨ Key Features

- **90-Day Velocity:** Visual progress bars tracking your journey to job readiness.
- **Task Tracker:** Manage daily coding tasks and to-dos with status updates.
- **Learning Analytics:** Log study hours and platforms (Udemy, YouTube, etc.) with dynamic Chart.js visualizations.
- **Glassmorphism UI:** A modern, dark-themed interface built for focus and performance.
- **Gamified Stats:** XP and Leveling system based on completed tasks.

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Database:** PostgreSQL (with `psycopg2`)
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript
- **Charts:** Chart.js
- **Auth:** Flask-Login (Secure Password Hashing)
